# Build and production preview

This project now supports a build + preview flow similar to frontend tooling:

- `bun run build`: creates a `dist/` folder and minifies JS/CSS assets from `public/`
- `bun run preview`: runs the built app from `dist/` in production mode
- `bun run start`: same as preview (starts built output)
- `bun run start:dev`: runs the source app directly (development mode)

Recommended production run locally:

1. `bun run build`
2. `bun run preview`
